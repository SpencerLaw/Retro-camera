# 哆啦A梦教室分贝监测仪

## 简介
这是一个实时监测环境噪音的应用，当噪音超过设定阈值时会自动触发警报。使用可爱的哆啦A梦形象，适合教室、自习室等需要保持安静的环境。

## 功能特性

### 核心功能
- 📊 实时监测环境分贝
- 🔔 可自定义报警阈值（40-90 dB）
- ⚠️ 超过阈值自动警报（视觉+声音）
- ⏱️ 安静时长计时
- 📈 警告次数统计
- 🌓 深色/浅色模式切换

### 交互特性
- 哆啦A梦表情会根据噪音情况变化：
  - 😊 安静状态：开心表情
  - 😰 预警状态：担忧表情
  - 😡 报警状态：生气表情 + 抖动动画
- 超大字体实时显示当前分贝
- 可视化音量条显示
- 阈值线标记

### 安全特性
- 屏幕常亮（防止监测中断）
- 麦克风断开自动检测
- 友好的权限申请引导

## 技术实现

### 核心技术
- React Hooks
- Web Audio API
- MediaDevices API
- Canvas API（用于分贝计算）
- Wake Lock API（屏幕常亮）

### 响应式设计
- 支持横屏和竖屏
- 适配手机、平板、电脑
- 触摸友好的UI设计

## 使用说明

1. 点击哆啦A梦卡片进入应用
2. 点击"开始安静 自习开始！"按钮
3. 允许浏览器访问麦克风
4. 应用开始实时监测环境噪音
5. 可以通过底部滑块调整报警阈值
6. 可以随时重置警告次数

## 样式特点

### 哆啦A梦配色
- 蓝色主题：#1293EE (哆啦A梦蓝)
- 红色警告：#DD0000 (报警红)
- 黄色铃铛：#FACE05 (铃铛金)
- 深色模式：#0A192F

### 动画效果
- 报警时的抖动动画
- 背景颜色脉冲动画
- 巨型警告文字
- 平滑的数值过渡

## 注意事项

1. 需要浏览器支持 Web Audio API
2. 必须允许麦克风权限
3. 建议在 HTTPS 环境下使用
4. 分贝值为估算值，非专业级精确测量
5. Chrome 在本地文件可能限制麦克风访问，建议使用 Edge 或部署到服务器
